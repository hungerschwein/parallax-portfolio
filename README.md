# My Portfolio

## Overview
My modern, scroll-animated, parallax portfolio website

## Installation
**Clone the repository**
```bash
git clone https://github.com/hungerschwein/parallax-portfolio.git
cd portfolio
```

**Serve locally with Docker**

```bash
docker-compose up -d
```
Then visit `http://localhost:8080`.
